| 字段          | 是否必填 | 描述                     | 默认值      |
| :------------ | :------: | :----------------------- | :---------- |
| owner         |   true   | 实际的仓库所有者         |             |
| repo          |   true   | 仓库名                   |             |
| token         |   true   | GITHUB_TOKEN             |             |
| ignore        |  false   | 屏蔽关键字 字符串用,隔开 | ""          |
| updatePath    |  false   | 更新文件地址             | 'README.md' |
| commitMessage |  false   | 更新提交内容             | '更新目录'  |
